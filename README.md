🚀 MENTARA 
Mentara is a next-gen, context-aware AI platform built to empower developers and entrepreneurs. It streamlines the chaotic journey from "idea phase" to "system design" and "venture roadmap" generation. 🧠✨

💡 The Vision
Mentara acts as your Personal Venture Mentor. It doesn't just write code; it understands the architecture, the business logic, and the scalability requirements of your startup ideas.

🛠 Tech Stack
Frontend: Next.js (React), Tailwind CSS

Backend: Python, FastAPI, Uvicorn

AI Engine: Google Gemini API, Cognee (Context Management)

Database: Supabase / Local JSON

🌟 Key Features
Venture Roadmap Generation: Get step-by-step milestones for your startup.

Intelligent System Design: Automated feedback on your project architecture.

Context-Aware Mentoring: Powered by Cognee to remember your project's unique history and goals.

Emergency SOS: Built-in safety features for developer peace of mind.

⚙️ Local Setup
Clone the Repo:

Bash
git clone https://github.com/YashYadav152006/MENTARA.git
cd MENTARA
Backend Configuration:

Navigate to backend/.

Create a virtual environment: python -m venv venv.

Activate it and run pip install -r requirements.txt.

Security Notice: This repo does not contain API keys. Please add your GEMINI_API_KEY in your environment files (app_cognee.py or backend1.env).

Run the Project:

Backend: python -m uvicorn main:app --reload

Frontend: npm run dev

🛡 Security First
API keys have been excluded from this repository for security purposes. Please configure your local environment variables before running the application.
